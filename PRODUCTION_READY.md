## Exact command to run the production batch

```bash
cd C:\Users\Mantas\Desktop\directory-submission-skill
python scripts/batch_submit.py --profile product-profile.yaml --queue queue.yaml --priority high
```

To dry-run the entire queue first (recommended, costs nothing, takes ~4 hours wall-clock):

```bash
python scripts/batch_submit.py --profile product-profile.yaml --queue queue.yaml --dry-run
```

To run a single priority bucket:

```bash
# 27 high-priority targets, ~27 minutes wall-clock
python scripts/batch_submit.py --profile product-profile.yaml --queue queue.yaml --priority high
```
